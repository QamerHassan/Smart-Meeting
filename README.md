# Smart Team Productivity Web App

A **web application to help teams manage meetings, tasks, and productivity intelligently**. This app leverages AI/NLP, real-time updates, and analytics to optimize team collaboration and efficiency.

---

## Features

### 1. Meeting Scheduler
- Suggests optimal meeting times based on participantsâ€™ calendars.
- Detects scheduling conflicts and proposes alternatives.

### 2. Task Auto-Assignment
- Automatically creates tasks from meeting notes using NLP (keyword extraction).
- Assigns tasks to relevant team members.

### 3. Productivity Insights
- Tracks time spent in meetings vs productive hours.
- Provides visual dashboards highlighting productivity bottlenecks.

### 4. Reminders & Notifications
- Sends Slack/email notifications for upcoming meetings, pending tasks, and deadlines.

---

## Tech Stack

- **Frontend:** Next.js + Tailwind CSS
- **Backend:** .NET 8 or Node.js + PostgreSQL
- **AI/NLP:** Python + FastAPI for processing meeting notes
- **Real-time Updates:** WebSockets or SignalR
