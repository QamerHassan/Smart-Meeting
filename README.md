## 🌟 About the Project

**Smart Meeting** is a next-generation, AI-driven web application engineered to transform the way modern teams plan, execute, and learn from their meetings. Gone are the days of scheduling chaos, forgotten action items, and untracked productivity. With Smart Meeting, every interaction becomes a strategic opportunity.

Powered by cutting-edge **Natural Language Processing (NLP)**, real-time communication protocols, and intelligent automation, Smart Meeting seamlessly integrates into your team's workflow — extracting tasks, optimizing schedules, and delivering visual analytics that hold your team accountable and keep productivity soaring.

Whether you're a growing startup or an enterprise team, **Smart Meeting is the intelligent backbone your collaboration deserves**.

---

## ✨ Key Features

### 🗓️ AI-Powered Meeting Scheduler
- Suggests **optimal meeting times** based on participants' live calendar availability.
- Proactively detects scheduling conflicts and proposes smart alternatives.
- Eliminates back-and-forth coordination with fully automated time recommendations.

### 🤖 Intelligent Task Auto-Assignment
- Leverages **NLP (keyword & intent extraction)** to automatically generate actionable tasks directly from meeting notes.
- Assigns tasks to the most relevant team members based on context and expertise.
- Ensures no follow-up slips through the cracks — ever.

### 📊 Productivity Insights & Analytics
- Tracks time spent in meetings versus focused, productive work hours.
- Delivers **interactive visual dashboards** that spotlight bottlenecks, peak performance windows, and team efficiency trends.
- Empowers leadership with data-driven decisions and actionable insights.

### 🔔 Smart Reminders & Notifications
- Integrates with **Slack and email** to deliver timely notifications for upcoming meetings, pending tasks, and approaching deadlines.
- Keeps every team member aligned, informed, and on schedule — automatically.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/QamerHassan/Smart-Meeting.git
cd Smart-Meeting
```

### 2. Install Frontend Dependencies
```bash
cd meeting-assistant-frontend
npm install
npm run dev
```

### 3. Start the Backend
```bash
cd meeting-assistant-backend
# Follow backend-specific setup instructions
```

### 4. Run the NLP Service
```bash
cd nlp-service
pip install -r requirements.txt
uvicorn main:app --reload
```
